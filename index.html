Install Axon

curl -sSL axon.mlosfoundation.org | sh
2

Install a Model

Choose from HuggingFace, PyTorch Hub, TensorFlow Hub, or ModelScope

axon install hf/bert-base-uncased@latest

Load Kernel Module
